# GitHub Update Server Setup

To make ByteNet auto-update using your GitHub repository, follow these steps:

## 1. Upload this folder
Upload the `release_system` folder to your GitHub repository (e.g., `https://github.com/YOUR_USERNAME/YOUR_REPO`).

## 2. Get the Raw URL
1. Go to the `version` file on GitHub.
2. Click the **"Raw"** button.
3. The URL should look like: `https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/release_system/version`

## 3. Update constants.py
Copy the URL **WITHOUT** the final `/version` part and paste it into `desktop/core/constants.py`:

```python
# desktop/core/constants.py
VERSION = "1.7.4"
UPDATE_URL = "https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/release_system"
```

## 4. Re-compile ByteNet
Build your EXE again. Now, whenever you want to push an update:
1. Upload your new EXE to GitHub Releases.
2. Update the `version` file in this folder with the new version number and the download link.
3. Users will automatically get the update prompt next time they open the app!
